本指南提供了如何进入ZITADEL Cloud 客户门户网站的快速起点，您可以在那里管理您的 ZITADEL实例。 在获取信息时关注所有指南以了解本节，以创建您的第一个实例、升级到付费等级并连接您的第一个客户端。

## 尝试ZITADEL Cloud

1. 访问 [zitadel.cloud](https://zitadel.cloud/) 创建您的帐户。 如果您已经有一个ZITADEL实例与您的客户门户用户登录。
2. 输入创建组织的数据(名字、姓氏、电子邮件、用户名和组织名称)
3. 点击"让我们开始"，我们将为您创建一个用户。
4. 您将收到验证电子邮件以验证客户门户的用户
5. 使用新创建的用户登录到自定义门户网站

![客户门户登陆页面](/img/manuals/portal/customer_portal_landing_page.png)

## 登录到客户门户网站

使用您的客户门户网站登录ZITADEL客户门户。 这里您可以管理所有不同的实例、订阅和计费数据。

1. 转到 [zitadel.cloud](https://zitadel.cloud)
2. 点击登录
3. 使用您的 ZITADEL Cloud用户

了解更多关于客户门户网站 [的信息](/docs/guides/manage/cloud/overview)。

## 创建一个新实例

:::note
这发生在 [ZITADEL Cloud 客户门户](https://zitadel.cloud) :
:::

一个实例的创建和管理在客户门户中进行。 要管理您现有的实例您需要与客户门户用户登录。 请注意，此用户与实例本身不同。

1. Click ![新实例](/img/manuals/portal/customer_portal_new_instance.gif)

## 登录到您的实例

:::note
这发生在您的 ZITADEL Cloud实例控制台
:::

在您初始化了新创建的 ZITADEL实例的第一个管理员用户。 您可以访问实例的控制台来管理您的所有资源。 要使用用户登录，您已初始化。 您将在用户初始化电子邮件或您实例的详细页面中找到访问您实例的链接 [ZITADEL Cloud ](https://zitadel.cloud)。

我们为 ZITADEL Cloud实例生成了一个独特的域名，这个域名看起来像这样： {InstanceName}-{RandomString}.zitadel.cloud
