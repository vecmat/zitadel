## ZITADEL DevOps

在本届会议上，您的第二级支持和业务团队将了解如何为技术支持问题和根源分析提取相关信息。 我们还将介绍我们的DevOps最佳做法，并回答你的问题。

**音频**: 二级支持员工, 操作 **持续时间**: 0.5天

**涵盖主题**:

- 事件类型
- 数据库方案和计算模型
- 正在访问数据库
- 观测能力(日志、错误、计量、跟踪)
- 业务最佳做法(部署、备份、联网等)
- 问题&A

## ZITADEL Administrator

在这次实践培训中，您的员工将获得一个完整的系统概览，并学习如何配置和使用ZITADEL。 您的支持员工将获得提供用户支持所需的知识，而您的解决方案所有者将获得对客户整合的理解。

**音频**: 1/2级支持员工, 解决方案所有者, QA Manager (可选)  
**持续时间**: 0.5天

**涵盖主题**:

- IAM Configuration
- 走访所有功能
- 用户 / 手册
- APIs
- 令牌的验证
- 客户整合最佳做法
- 问题&A
