您可以订阅不同的新闻通讯和通知。

## 更改通知/通讯设置

1. 点击右上角的用户头像
2. 在已经打开的菜单中，您可以看到点击“编辑通知”
3. 您可以看到所有不同的新闻通讯和通知，现在可以启用或禁用
