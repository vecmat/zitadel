## 故障排除

如果有些东西不起作用，请检查云端WF规则是否有可能发生违反情事。 这两条规则已知被触发：

- 100001 Anomaly:Header:User-Agent - 缺少云层专家
- 100004 Anomaly:Header:User-Agent, Anomaly:Header:Referer - 丢失或为空
