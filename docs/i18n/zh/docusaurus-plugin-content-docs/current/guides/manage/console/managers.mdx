要配置ZITADEL中的管理员，请到您想要添加的资源 (例如，实例、组织、项目、大型项目)。 在控制台的右侧部分，你可以在细节部分中finde **MANAGERS** 这里有一个当前管理员的列表，并可以添加一个新的管理员。

<img alt="管理人员" src="/img/guides/console/managers.png" width="200px" />

添加新管理器时，您可以选择多个角色，其中一些角色只允许读取数据。 如果您为您的项目之一添加服务用户，您只需阅读即可，这将特别有用。

每个默认情况下，您将只搜索所选组织中的用户。 如果您想要赋予机构外的用户角色，您需要切换到全局搜索并输入用户的确切登录名。 这将防止用户猜测其他组织的用户。

<img alt="管理人员" src="/img/guides/console/addmanager.png" width="390px" />

## 角色

| 名称          | 作用                               | 描述                                 |
| ----------- | -------------------------------- | ---------------------------------- |
| IAM 所有者     | IAM_OWNER                        | 管理IAM，管理所有带有其内容的组织                 |
| IAM 所有者查看器  | IAM_OWNER_VIEWER               | 查看内部管理并查看所有组织的内容                   |
| IAM 组织管理器   | IAM_ORG_MANAGER                | 管理所有组织，包括他们的政策、项目和用户               |
| IAM用户管理器    | IAM_USER_MANAGER               | 管理所有用户和他们对所有机构的授权                  |
| 矿石          | ONG_OWNER                        | 管理组织内的所有内容                         |
| 组织所有者查看器    | VIEWER_DESC                      | 查看组织内的所有内容                         |
| 组织用户管理器     | USER_MANAGER                     | 在组织内管理用户和他们的授权                     |
| Org 用户权限编辑器 | USER_PLAYLIST_TITLE            | 管理用户授予并查看此操作所需的一切功能                |
| Org 工程权限编辑器 | PROJECT_PERMISSION_DESCRIPTION | 将项目授予其他组织并查看实现此目标所需的一切内容           |
| Org 工程创建者   | RIG_PROJECT_CREATOR            | 这个角色用于全球组织中的用户。 他们可以创建和管理项目。       |
| 项目所有者       | PROJECT_OWNER                    | 管理项目内的一切。 这包括为项目提供用户。              |
| 项目所有者查看器    | PROJECT_OWNER_VIEWER           | 查看项目内的所有内容。                        |
| 项目所有者全局项目   | PROJECT_OWNER_GLOBAL           | 与PROJECT_OWNER相同，但在全球组织中。          |
| 项目所有者查看器    | PROJECT_OWNER_VIEWER_GLOBAL    | 与PROJECT_OWNER_VIEWER相同，但在全球组织中。 |
| 项目所有者       | PROJECT_GRANT_TITLE            | 与PROJECT_OWNER 相同，但是获得授权的程序相同。     |


## 配置角色

如果您运行了自己托管的 ZITADEL距离，您可以通过覆盖默认值来定义您的自定义角色。 aml 在 InternalAuthZ 版块中，您将找到所有角色和哪些权限。

示例：
```bash
InternalAuthZ:
  RolePermissionMappings:
    - Role: "IAM_OWNER"
      Permissions:
        - "iam.read"
        - "iam.write"
```
