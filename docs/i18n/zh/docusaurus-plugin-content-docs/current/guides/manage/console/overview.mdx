## 什么是控制台？

控制台是您实例的仪表盘界面。 它可以从客户端门户定义的所有配置实例域中访问。 控制台用于配置全局实例设置，可以被多个管理员使用。 在这里阅读更多关于 [控制台管理器](./managers)

您的应用程序用户也可以使用它来修改他们的个人资料，尽管我们建议您建立自己的用户界面。

登录后，您会收到主页的问候。 此页面允许您设置设置设置和项目的快捷方式。 控制台在您设置当前机构的 **左上端** 上有一个上下文切换器。 根据您的情况，可以创建多个组织 (B2B) 或者您可以坚持到您的全球组织 (B2C)。 要了解您的使用情况和我们建议如何设置您的组织，请阅读 [解决方案](../../solution-scenarios/introduction) 指南。
